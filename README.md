# SkillForge Project

## Project Description
This is a SkillForge project aimed at providing an interactive educational platform for students and instructors, leveraging modern web technologies.

## Technologies Used
- **Frontend**: Angular 17, RxJS
- **Backend**: Laravel 10, WebSocket

## Installation Steps
### Frontend
1. Clone the repository.
2. Change directory to `frontend`.
3. Install dependencies by running `npm install`.

### Backend
1. Clone the repository.
2. Change directory to `backend`.
3. Install dependencies by running `composer install`.

## WebSocket Setup Instructions
1. Ensure the WebSocket server is running.
2. Configure your WebSocket client using socket.io.

## Build and Run Commands
- **Frontend**: `ng serve` to start the Angular development server.
- **Backend**: `php artisan serve` to start the Laravel development server.
